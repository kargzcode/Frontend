Frontend Development

Frontend Development is the development of the graphical user interface of a website through the use of HTML, CSS, and JavaScript, so that the user can view and interact with that specific website.

  • HTML: The hypertext markup language or HTML is a standard markup language for documents designed to be displayed by the browser.

  • CSS: Cascading Style Sheets CSS is a stylesheet language we can use to style (color, layout, etc.) the structure created by HTML.


  1. Pure CSS

  2. Bootstrap (framework)


  3. Tailwind (framework)


  4. Sass (preprocessor)


  • JAVASCRIPT: The hypertext markup language or JS is a standard markup language for documents designed to be displayed by the browser.



    1. Pure CSS

    2. Angular-JS (framework)


    3. Vue(framework)


    4. Backbone-JS (preprocessor)


    5. RIOT(preprocessor)
